# КОД СМОТРИТЕЛЯ (Watcher Code)

Лендінг-сайт для продукту "Код Смотрителя" - містичний цифровий артефакт для самопізнання та трансформації.

## Технології

- **Next.js 16** - React фреймворк з App Router
- **React 19** - UI бібліотека
- **TypeScript** - типізація
- **Tailwind CSS 4** - стилі
- **Framer Motion** - анімації
- **Lucide React** - іконки

## Запуск локально

```bash
# Встановити залежності
npm install

# Запустити dev сервер
npm run dev

# Відкрити http://localhost:3000
```

## Деплой на Vercel

### Автоматичний деплой

1. Підключіть репозиторій GitHub до Vercel
2. Vercel автоматично визначить Next.js проєкт
3. При кожному push в `main` буде автоматичний деплой

### Ручний деплой

```bash
# Встановити Vercel CLI
npm i -g vercel

# Задеплоїти
vercel
```

## Структура проєкту

```
├── src/
│   ├── app/          # Next.js App Router
│   └── components/   # React компоненти
├── public/           # Статичні файли
│   ├── audio/        # Аудіо файли
│   └── *.jpg, *.png  # Зображення
└── package.json
```
